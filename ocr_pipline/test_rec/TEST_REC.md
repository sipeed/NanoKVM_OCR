# test_rec - OCR 识别测试程序（带字符详情）

## 功能增强

现在 `test_rec` 程序会输出每个字符的详细识别信息，包括：

### 输出示例

```
========================================
Recognition Result:
========================================
Text: 以及 ATX 开关机、复位按键
Confidence: 0.9234
========================================

Character Details:
------------------
Total characters: 13

Idx  Char  Hex     Conf%   Index
---  ----  ------  -------  -----
  0  '以'   0xE4    95.23%    234
  1  '及'   0xE5    93.45%    567
  2  ' '   0x20    98.12%     12
  3  'A'   0x41    99.87%     45
  4  'T'   0x54    99.65%     67
  5  'X'   0x58    99.23%     89
  6  '开'   0xE5    94.56%    789
  7  '关'   0xE5    92.34%    890
  8  '机'   0xE6    96.78%    901
  9  '、'   0xE3    91.23%     34
 10  '复'   0xE5    93.67%   1012
 11  '位'   0xE4    94.89%   1123
 12  '按'   0xE6    95.45%   1234
 13  '键'   0xE9    96.12%   1345
------------------

Statistics:
  Max confidence: 99.87%
  Min confidence: 91.23%
  Avg confidence: 95.43%

Result saved to: test.jpg.txt
```

### 输出字段说明

- **Idx**: 字符索引（从 0 开始）
- **Char**: 字符本身（如果是可打印字符）
- **Hex**: 字符的十六进制编码
- **Conf%**: 该字符的置信度（百分比）
- **Index**: 字符在字典中的索引

### 统计信息

- **Max confidence**: 最高置信度
- **Min confidence**: 最低置信度
- **Avg confidence**: 平均置信度

### 保存的文件内容

输出的 `.txt` 文件也会包含所有字符的详细信息，方便后续分析。

## 使用方法

```bash
cd build_ax630c/install/bin
./test_rec test.jpg ../../models/ocr_rec.axmodel ../../models/dict.txt
```

## 应用场景

1. **调试识别错误**：查看哪些字符置信度低
2. **模型评估**：统计整体识别质量
3. **字典分析**：查看字符在字典中的分布
4. **质量监控**：设置置信度阈值，过滤低质量结果
